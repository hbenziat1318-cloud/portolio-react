#  Portfolio React Professionnel

Un portfolio moderne et responsive développé avec React, Vite et TypeScript.

<br>
https://portolio-react-ten.vercel.app/
<br>

## Installation et Démarrage

### Prérequis
- Node.js (version 16 ou supérieure)
- npm ou yarn

### Installation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/hbenziat1318-cloud/portolio-react.git
   cd portolio-react
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   # ou
   yarn install
   ```

3. **Démarrer en mode développement**
   ```bash
   npm run dev
   # ou
   yarn dev
   ```

4. **Ouvrir dans le navigateur**
   ```
   http://localhost:5173
   ```
